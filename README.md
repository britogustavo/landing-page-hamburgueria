# 🍔 Landing Page - Hamburgueria

🚧 Projeto em desenvolvimento 🚧  

Este projeto consiste no desenvolvimento de uma landing page para uma hamburgueria fictícia, com foco na prática de desenvolvimento web utilizando HTML e CSS.

---

## 📌 Status do projeto

Atualmente em fase inicial de desenvolvimento.

### ✔️ Já implementado
- Estrutura inicial do projeto  
- Início da navbar  
- Estruturação inicial do layout

### 🚧 Em desenvolvimento
- Finalização da navbar  
- Seção principal (hero)  
- Estilização completa  

### ⏳ Próximos passos
- Adicionar seção de produtos  
- Melhorar o design  
- Implementar responsividade  
- Implementação futura de interatividade com JavaScript

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  

## 🎨 Ferramentas

- Canva (criação do mockup)
- GIMP (otimização e compressão de imagens)



## 🎨 Design (Mockup)

Antes do desenvolvimento, foi criado um mockup da interface para planejar a estrutura e o visual da aplicação.
